# Installation

## Igor Pro

1. Install Igor Pro
2. Install the HDF5 package in Igor Pro
3. Create aliases/shortcuts for the following IPF files.


DisplayFigFromMatlab.ipf

ManookinLabIgorProcedures.ipf

4. Move these aliases to the Igor Procedures folder in your Documents directory (e.g., '~/Documents/WaveMetrics/Igor Pro 8 User Files/Igor Procedures'
5. Start Igor Pro
